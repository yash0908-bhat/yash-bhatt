# Hi, I'm Yash Bhatt

I am a Master of Applied Technologies student at Unitec Institute of Technology, New Zealand, with a research focus on LLM-based multi-agent systems, communication efficiency, and applied AI systems.

My current thesis investigates dynamic context-aware communication filtering for LLM-based multi-agent systems. The aim is to reduce redundant inter-agent communication, token usage, and latency while preserving task accuracy and coordination quality.

## Research Interests

- LLM-based multi-agent systems
- Communication efficiency in AI agents
- Semantic routing and message utility modelling
- Parameter-efficient fine-tuning, including LoRA
- Applied machine learning and experimental evaluation

## Current Thesis Project

**Dynamic Context-Aware Communication Filtering for LLM-Based Multi-Agent Systems**

This research compares:

- Full communication baseline
- Rule-based message utility control
- LoRA-based semantic routing

The project evaluates communication-performance trade-offs using task accuracy, token usage, latency, routing accuracy, redundancy reduction, and coordination quality.
